## 🚀 İlk Adımlar (Yarın)

### 1. Git Repository Hazırla
```bash
# .gitignore güncelle
echo ".env" >> .gitignore
echo "*.sqlite" >> .gitignore
echo "node_modules/" >> .gitignore

# Git init (eğer yoksa)
git init
git add .
git commit -m "Initial commit: Boykot Platform v1.0"
```

### 2. GitHub'a Yükle
```bash

# Sonra:
git remote add origin https://github.com/[username]/boykot-platform
git branch -M main
```
