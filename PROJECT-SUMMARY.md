# 🎯 Boykot Platform - Proje Özeti

## 📋 Proje Hakkında

**Boykot Platform**, vatandaşların sosyal sorumluluk kampanyaları başlatabildiği, oy verebildiği ve topluluk olarak etkileşime geçebildiği kapsamlı bir aktivizm platformudur.

### 🎨 Teknoloji Stack

**Backend:**
- Node.js + Express.js
- TypeScript
- SQLite (better-sqlite3)
- JWT Authentication
- bcrypt (şifreleme)

**Frontend:**
- Next.js 14 (App Router)
- React 18
- TypeScript
- Tailwind CSS
- Context API (State Management)

**Özellikler:**
- RESTful API (130+ endpoint)
- Responsive Design
- Dark/Light Mode
- Çoklu Dil Desteği (TR/EN)
- Real-time Updates

## 🔒 Güvenlik Özellikleri

1. **Authentication & Authorization**
   - JWT token bazlı kimlik doğrulama
   - Refresh token sistemi
   - Role-based access control (RBAC)

2. **Data Protection**
   - bcrypt ile şifre hashleme (12 rounds)
   - SQL injection koruması (prepared statements)
   - XSS koruması
   - CSRF koruması

3. **Anti-Bot & Spam**
   - reCAPTCHA v3
   - Device fingerprinting
   - Rate limiting
   - IP tracking

4. **Privacy**
   - Anonim oy verme
   - GDPR uyumlu veri yönetimi
   - Kullanıcı verisi şifreleme
