## 📋 PROJE HAKKINDA

**Boykot Platform**, vatandaşların sosyal sorumluluk kampanyaları başlatabildiği, oy verebildiği, imza toplayabildiği ve topluluk olarak etkileşime geçebildiği kapsamlı bir aktivizm platformudur.

**Proje Durumu:** ✅ %100 Tamamlandı - Production Ready  
**Geliştirme Süresi:** ~7 saat (2 Şubat 2026)  
**Toplam Özellik:** 30+ sistem  
**Kod Satırı:** ~15,000+  
**API Endpoints:** 130+

## 🎨 TEKNOLOJİ STACK

### Backend
- **Runtime:** Node.js v18+
- **Framework:** Express.js 4.x
- **Language:** TypeScript 5.x
- **Database:** SQLite (better-sqlite3)
- **Authentication:** JWT (jsonwebtoken)
- **Password Hashing:** bcrypt (12 rounds)
- **Validation:** express-validator
- **File Upload:** multer (5MB limit)
- **Email:** nodemailer (SMTP ready)
- **Logging:** winston
- **Security:** helmet, cors
- **Documentation:** swagger-ui-express

### Frontend
- **Framework:** Next.js 14.x (App Router)
- **Library:** React 18.x
- **Language:** TypeScript 5.x
- **Styling:** Tailwind CSS 3.x
- **State Management:** Context API
- **HTTP Client:** Fetch API
- **Forms:** React Hooks
- **Routing:** Next.js App Router

### Development Tools
- **Package Manager:** npm
- **Code Editor:** VS Code
- **Version Control:** Git
- **API Testing:** Swagger UI
- **Database Tool:** SQLite Browser
